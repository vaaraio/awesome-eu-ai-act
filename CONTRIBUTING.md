# Contributing to Awesome EU AI Act

Thank you for helping make this the best EU AI Act resource list on the internet. All contributions are reviewed by [Carlos Hernandez](https://github.com/carloshvp) within 48 hours.

## How to add a resource

1. **Fork** this repository
2. **Edit** `README.md` — add your resource to the most relevant section
3. **Open a Pull Request** with a clear title, e.g. `Add: [Resource Name]`

### Entry format

```markdown
- [Resource Name](https://url.example.com) — One-sentence description of what it is and why it's useful.
```

Keep descriptions concise (under 100 characters) and factual — no marketing language.

## What belongs here

- Directly relevant to EU AI Act compliance, understanding, or implementation
- Publicly accessible (paywalled resources are accepted if clearly noted)
- Working links (we run automated link checks)
- Not already listed

## What does not belong here

- Resources unrelated to EU AI Act or AI governance
- Duplicate entries
- Personal blogs without substantial original analysis
- Promotional content without genuine educational value

## One resource per PR

Smaller PRs are reviewed and merged faster. If you have 5 resources to add, open 5 separate PRs or group them by section in a single PR with a clear description.

## Broken links

Please [open an issue](https://github.com/GenAI-Gurus/awesome-eu-ai-act/issues/new?template=broken-link.yml) using the broken-link template. We check links automatically each week, but human reports help us fix things faster.
